# Загрузка Android-проекта в новый репозиторий

1. Создайте пустой публичный репозиторий без README, `.gitignore` и лицензии.
2. Распакуйте архив приложения на компьютере.
3. Загрузите в корень репозитория всё содержимое распакованной папки.
4. Проверьте наличие папок `lib`, `assets`, `test`, `tool` и файла `pubspec.yaml`.
5. Проверьте наличие `.github/workflows/build_android.yml`.
6. Если скрытая папка `.github` не загрузилась, создайте файл `.github/workflows/build_android.yml` через интерфейс GitHub и вставьте содержимое `WORKFLOW_BUILD_ANDROID.yml`.
7. Откройте `Actions` → `Build Android APK` → `Run workflow`.
8. После успешной сборки скачайте артефакт `ascona-scanner-android-v1`.
9. Распакуйте артефакт и установите `app-release.apk` на телефон.

Версия 1.0 работает в строгом порядке: сначала код заказа, затем штрихкоды товаров. Наименование подставляется из встроенного локального справочника.
